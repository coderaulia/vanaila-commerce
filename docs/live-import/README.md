# Live Import Payloads

This folder contains JSON payload snapshots used for controlled content imports and merge/replace operations.

These files are operational artifacts, not general setup documentation:

- `01-pages-product-hris.merge.json`
- `02-portfolio-image-updates.merge.json`
- `03-settings-navigation.patch.json`
- `04-settings-from-local.replace.json`

Before applying one of these payloads, confirm the target environment, back up current CMS content, and run the import flow in a non-production environment first.
